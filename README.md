## Guía para desarrollo <a name="3"></a>

Este proyecto requiere **[Node.js](https://nodejs.org/)** para su desarrollo.

```
git clone https://github.com/DiazLeonel/Proyecto-Real-PCentriX.git
```
```
cd Proyecto-Real-PCentriX
```
```
npm install
```
```
npm run start
```
